# ameshop

### 1. ใส่ลิงค์ใน link.txt จากนั้นเปิด Command Line และรัสโค้ดต่อไปนี้เพื่อเก็บข้อมูล
```
node getProductInfo.js
```

### 2. เมื่อเก็บข้อมูลเรียบร้อย ใน Command Line ให้รันคำสั่งต่อไปนี้เพื่อเปิด Server
```
node server.js
```

### 3. เข้าเว็บ http://localhost:3000 เพื่อรันโปรแกรม
```
http://localhost:3000
```
